# Books-MERN-stack
Books MERN stack with frontend and backend <br>
Frontend: ReactJS <br>
Backend: Nodejs, Express <br>
Database: MongoDB <br>

CRUD operation: <br>
Create record of books <br>
Read/View the list of books in the database + Search  <br>
Update/Edit the properties of books <br>
Delete any record <br>
